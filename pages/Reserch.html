Our Research

Our research is focused on the following areas:

  • Artificial Intelligence
  • Machine Learning
  • Computer Vision
  • Robotics
  • Human Computer Interaction

Active Research Projects

Our current research projects are:

Robots for the Home

Robots for the Home is a project that
aims to develop robots that can help people
in their homes.
The robots will be able to perform tasks
such as cleaning, cooking, and taking care of pets.

Artificial Intelligence and Machine Learning

Artificial Intelligence and Machine Learning is a project
that aims to develop artificial intelligence and machine
learning algorithms that can be used in various applications.
The algorithms will be able to learn from
data and make predictions about the future.

Human Computer Interaction

Human Computer Interaction is a project that aims
to develop new ways of interacting with computers.
The project will explore the use of gestures,
voice, and other forms of input to interact
with computers.


Previous Research Projects

Our previous research projects are:

Micro-services and Data-Sharing Cyber-infrastructure

Micro-services and Data-Sharing Cyber-infrastructure
is a project that aims to develop a micro-services
and data-sharing cyber-infrastructure for the
Internet of Things.
The project will develop a platform that can be
used to share data between devices and services.

Analysis of Emotional Parameter of Learning Cybersecurity with Games and Training using non-invasive IoT

Analysis of Emotional Parameter of Learning Cybersecurity
with Games and Training using non-invasive
IoT is a project that aims to develop a system
that can be used to analyze the emotional
parameter of learning cybersecurity with games and training
using non-invasive IoT.
The project will develop a system that can be
used to analyze the emotional parameter of learning
cybersecurity with games and training using non-invasive IoT.

Development of a Smart Home System for the Elderly

Development of a Smart Home System for the
Elderly is a project that aims to develop
a smart home system for the elderly.
The project will develop a system that can be
used to monitor the health of the elderly and
provide assistance in case of emergency.

R-Mon: mHealth tool for Real-time Pulmonary Disease Monitoring

R-Mon: mHealth tool for Real-time Pulmonary Disease Monitoring
is a project that aims to develop a mHealth tool
for real-time pulmonary disease monitoring.
The project will develop a system that can be
used to monitor the health of the elderly and
provide assistance in case of emergency.
